# CrossFlow 🚦

**CrossFlow** - это информационная система для планирования графика дежурств сотрудников и учёта трафика на перекрёстках. Система позволяет менеджерам назначать дежурства, сотрудникам фиксировать данные о трафике, а также формировать отчёты с интеграцией с Google сервисами.

## 🎯 О проекте

**CrossFlow** разработан для автоматизации процесса наблюдения за трафиком на перекрёстках. Система позволяет:

- 📊 Фиксировать данные о загруженности перекрёстков с 5-минутным интервалом
- 👥 Управлять пользователями с двумя ролями (менеджер и сотрудник)
- 🗺️ Работать с интерактивными картами для определения зон дежурств
- 📈 Формировать детализированные отчёты с экспортом в Google Docs
- 🔄 Интегрироваться с Google Calendar для напоминаний о дежурствах
- 📝 Импортировать результаты дежурств из Google Sheets

## ✨ Функциональные возможности

### Для менеджера:
- ✅ Подтверждение регистрации новых сотрудников
- ✅ Создание и редактирование перекрёстков на интерактивной карте
- ✅ Добавление направлений движения для каждого перекрёстка
- ✅ Назначение дежурств сотрудникам с указанием времени и зоны
- ✅ Просмотр и фильтрация дежурств по датам
- ✅ Генерация отчётов по загруженности перекрёстков
- ✅ Экспорт отчётов в Google Docs

### Для сотрудника:
- ✅ Просмотр своих дежурств
- ✅ Запись данных о трафике (5-минутные интервалы)
- ✅ Фиксация количества ожидающих автомобилей по направлениям
- ✅ Отправка результатов дежурств через Google Sheets

### Интеграции:
- 🔐 **Google OAuth2** - аутентификация пользователей
- 📅 **Google Calendar** - создание событий с напоминаниями
- 🗺️ **OpenStreetMap (Leaflet)** - интерактивные карты для зон
- 📊 **Google Sheets** - импорт результатов дежурств
- 📄 **Google Docs** - экспорт отчётов
